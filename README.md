## Задание
- Изучить предоставленный код на Java, понять по коду какую задачу решал разработчик.

### Requirements
- PHP >= 8.3
- Composer >= 2
- make >= 4

### Installation
> `git clone git@github.com:NONstop5/omcode_test.git`
>
> `make install`

### Launch
> `make run`
